Privacy Policy

MyQBUI - Remote Control for qBittorrent
Last Updated: January 12, 2026

Overview

MyQBUI is a remote control application for qBittorrent. This privacy policy explains how the app handles your information.

Information We Collect

User-Provided Information

  • Server Connection Details: qBittorrent server URL, username, and password
  • Preferences: Download path history, card display preferences, notification settings

Automatically Collected Information

None

This app does not collect analytics, usage statistics, or any telemetry data

How We Store Your Information

Local Storage Only

All data is stored exclusively on your device:

  • Server credentials are encrypted and stored in Android's secure storage
  • Biometric authentication (if enabled) uses Android's native Keystore
  • Preferences are stored in local SharedPreferences
  • Path history is saved in local storage

No Cloud Backup

Your credentials and settings are not backed up to any cloud service. If you uninstall the app or reset your device, this data will be lost.

How We Use Your Information

  • Server Connection: Your credentials are used solely to authenticate with your qBittorrent server
  • Background Monitoring: When enabled, the app periodically checks your server to send notifications
  • Local Notifications: The app sends notifications for torrent events (completion, errors, pauses)

Third-Party Services

None

This app does not use any third-party services, including:

  • No analytics (Google Analytics, Firebase, etc.)
  • No crash reporting
  • No advertising networks
  • No data sharing with external parties

Network Communication

The app communicates only with:

  • Your qBittorrent server (URL you provide)
  • No other servers or services

Permissions

Required Permissions

  • Internet: To connect to your qBittorrent server
  • Notifications: To send torrent status notifications
  • Biometric (optional): For fingerprint/face unlock authentication

Optional Permissions

  • Background Work: For background monitoring feature (can be disabled)

Data Security

  • Credentials are encrypted using Android's Keystore
  • All communication with your server uses the protocol you configure (HTTP/HTTPS)
  • No data is transmitted to third parties

Your Rights

You have full control over your data:

  • Delete all data by uninstalling the app
  • Disable background monitoring at any time
  • Clear credentials by logging out

Children's Privacy

This app does not knowingly collect information from children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted within the app.
